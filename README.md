ğŸ“ Abstract

The Jewellery Store Web Application is a Flask-based e-commerce system designed to simplify online jewellery shopping.
It provides customers with an interactive interface to browse jewellery categorized under Gold, Silver, Platinum, etc., while the admin manages products, categories, orders, and users through a secure dashboard.

This system reduces manual effort, enhances accessibility, and ensures secure transactions with authentication and CSRF protection.
